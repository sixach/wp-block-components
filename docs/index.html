BackgroundSizeControl

src/background-size-control/index.js

UI controls that are designed to provide ability to adjust CSS background size properties. There are four different syntaxes you can choose from with this control: the keyword syntax ("auto", "cover", "contain" and "custom").

BackgroundSizeControl(props: Object): JSX.Element
Since: 1.0.0
Parameters
props (Object) The props that were defined by the caller of this component.
Name Description
props.id string The id of the element to which labels and help text are being generated.
props.label string Label shown before the spinner.
props.hideLabelFromVision boolean Whether to accessibly hide the label.
props.help boolean Optional help text for the control.
props.sizes Array (default sizesTable) Optionally override background position options.
props.className string The class that will be added with “components-background-size” to the classes of the wrapper div.
props.value Object Value of the control.
props.onChange Function Handle changes.
Returns
JSX.Element: UI controls.
Example
<BackgroundSizeControl
	label={ __( 'Background Size' ) }
	value={ backgroundSize }
	onChange={ ( value ) => setAttributes( { backgroundSize: value } ) }
/>

// => Object { selection: "custom", width: "15%", height: "12%" }

GenerateSvgPaths

src/generate-svg-paths/index.js

Generates corresponding HTML Path elements and enables the shape to be drawn.

Since: 1.0.0
Parameters
props (Object) The props that were defined by the caller of this component.
Name Description
props.paths Array List of SVG path strings.
props.attributes Object Object of HTML attributes used inside the opening tag to control the element’s behaviour.
Returns
JSX.Element: Corresponding HTML Path elements.
Example
const path = { d: 'M649.97 0L550.03 0 599.91 54.12 649.97 0z', attrs: { opacity: '0.5' } };
<GenerateSvgPaths path={ path } attributes={ { style: { fill: '#BADA55' } } } />

InnerHTML

src/inner-html/index.js

Component used as equivalent of Fragment with unescaped HTML, in cases where it is desirable to render dangerous HTML without needing a wrapper element. To preserve additional props, a div wrapper will be created if any props aside from children are passed.

InnerHTML(props: Object): JSX.Element
Since: 1.0.0
Parameters
props (Object) The props that will be passed through to the wrapper element.
Name Description
props.tagName string (default 'div') The tag name of the wrapper element.
props.children string Children should be a string of HTML.
props.props ...any
Returns
JSX.Element: Dangerously-rendering component.
Example
const content = '<span class=\"amount\"><bdi><span class=\"currency-symbol\">&pound;<\/span>11.05<\/bdi><\/span>';
<InnerHTML tagName="span">{ content }</InnerHTML>
